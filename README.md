# Terminal Quest

Interaktywna gra RPG ucząca obsługi terminala macOS/Linux. Wpisujesz prawdziwe komendy bezpośrednio w grze — ona je weryfikuje, uruchamia i pokazuje wynik. Uczysz się walcząc z bossami.

## Wymagania

- macOS (zsh)
- Python 3.10+
- Homebrew

## Instalacja

```bash
pip3 install rich --break-system-packages
```

## Uruchomienie

```bash
python3 game.py
```

## Jak grać

- **Enter** — następny quest
- **m** — mapa świata
- **q** — wyjście
- `/hint` — pokaż oczekiwaną komendę
- `/skip` — pomiń krok
